# Doctor Repository Readme
## **Prerequisites:**

Before downloading the github repository locally, make sure you have the following installed:

* Java 21 (JDK 21)

* A Java-compatible IDE (IntelliJ IDEA, Eclipse, etc.)

* Windows OS (recommended for Bluetooth compatibility)


#### **Dependencies:**

* junit:junit:RELEASE — JUnit testing framework

* org.junit.jupiter:junit-jupiter:RELEASE — JUnit Jupiter testing framework (part of JUnit 5)



## **Installation:**

### In case you want to run the project throught Intelij:

  1. Clone the repository locally

  2. Build the project by: mvn clean install

  3. Run the _UI_ class

### In case you only want to execute from the terminal:

  1. Download the _Doctor_Rep_jar_ folder located in the _out_ folder path

  2. Now through your terminal use _cd_ and _ls_ commands to navigate throught your directories until you reach _Doctor_Rep.jar_ file

  3. Finally run the application using the command: java -jar Doctor_Rep.jar
